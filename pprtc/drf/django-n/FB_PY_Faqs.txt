25 How to use  function inside class (@staticmethod)?
@staticmethod
We can use function inside class using @staticmethod
@staticmethod is similar to Python function but define in a class 
we can call function using object 

class MyCity:
    @staticmethod
    def print_city(city):
        return city
MyObj=MyCity()
MyObj.print_city("HYDERABAD")
